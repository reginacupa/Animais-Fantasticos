🐾 Animais Fantásticos
📚 Sobre o Projeto
Este projeto foi desenvolvido como parte do curso de JavaScript, com o objetivo de aplicar conceitos de manipulação do DOM, módulos JS e interatividade em uma página web. A aplicação apresenta uma lista de animais com informações dinâmicas, animações e contadores que são atualizados conforme a interação do usuário.
🧩 Funcionalidades
- Exibição de dados dinâmicos sobre cada animal
- Contador animado de números (quantidade de espécies, etc.)
- Navegação suave entre seções
- Menu responsivo e interativo
- Código modular com organização em arquivos JS separados
🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (ES6+)
- Webpack (para organização de módulos)
